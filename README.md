#  Smart University System

##  Overview
Smart University System is a web-based application that helps institutions manage student registration, fees tracking, and unit allocation efficiently.

---

##  Features
- Student Registration & Login
- Course-based Unit Display
- Unit Registration System
- Fee Progress Tracking (40% rule)
- QR/Scanner Verification System
- Admin Dashboard (Manage Students & Units)

---

##  Technologies Used
- PHP
- MySQL
- HTML & CSS
- JavaScript

---

##  Problem Solved
Many institutions experience delays and inefficiencies during unit registration.

This system:
- Reduces congestion
- Speeds up verification
- Ensures only eligible students register units
